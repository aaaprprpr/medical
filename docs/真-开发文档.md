## 当前患者接口

```http
GET    /api/patients
GET    /api/patients/{id}
POST   /api/patients
PUT    /api/patients/{id}
DELETE /api/patients/{id}
```

列表接口支持：

```text
keyword
gender
sortBy
order
```

示例：

```http
GET /api/patients?keyword=Patient&gender=男&sortBy=age&order=desc
```

## 下一步

患者信息前端接后端接口：

1. 打开患者信息页时请求 `GET /api/patients`。
2. 表格展示后端返回的患者数据。
3. 新增患者时调用 `POST /api/patients`。
4. 编辑患者时调用 `PUT /api/patients/{id}`。
5. 删除患者时调用 `DELETE /api/patients/{id}`。
6. 搜索面板变化后调用 `GET /api/patients?...`。
7. 表头点击排序时改变 `sortBy` 和 `order`。
