# MACE Risk Prediction in ARVC Patients via CMR: A Three-Tier Spatiotemporal Transformer with Pericardial Adipose Tissue Embedding
![The architecture of the proposed model, TTST](readme_img/TTST_architecture.png)

The model training consists of two stages：

- In the first stage, the IEPV module is trained.
- In the second stage, the parameters of IEPV are frozen, and the DSFI module is trained.

During IEPV training, two sets of parameters are optimized separately to extract slice-level features from both Cine and LGE images.

The train/test split is determined by the patient division in the Cine sequences.
The LGE, mask_Cine, and mask_LGE folders do not require manual train/test splits — the code will automatically locate the corresponding LGE and mask files based on each patient’s Cine sequence path.

## 2. Training IEPV
To train the IEPV module, run:
```
python train_Cine_img.py
python train_LGE_img.py
```

## 3. Training DSFI
To train the DSFI module, run:
```
python train_sequence.py
```

## 4. Prediction
After training, you can perform MACE risk prediction using:
```
python test_sequence.py
```

## Training Monitoring
You can monitor each training process via TensorBoard:
```
python -m tensorboard.main --logdir [logdir]
```
where `[logdir]` is automatically generated by the training scripts, for example: `python -m tensorboard.main --logdir logs/Cine_img_model`
